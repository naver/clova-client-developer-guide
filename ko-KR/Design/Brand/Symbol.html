심볼

CLOVA 심볼은 서비스의 핵심 가치를 나타내고 브랜드를 상징하는 중요한 요소입니다. 임의로 조정하거나 변경하면 안되며, 사용 시 최소 여백과 최소 크기를 유지해야 합니다. 이 페이지에서는 다음 내용을 다룹니다.

Note!

이 페이지에 첨부된 파일을 사용하기 전에 반드시 제휴 담당자에게 문의해주십시오.

기본 심볼

심볼을 적용할 때 일반적으로 기본 심볼을 사용하며 아래 이미지를 사용해야 합니다.

CLOVA 심볼의 기본 색상은 그라데이션을 적용합니다. 고유의 색상 구성을 변경할 수 없으며 지정된 RGB / CMYK 값을 유지해야 합니다.

기본 심볼 색상 코드

색상 구분 HEX 코드 RGB 값 CMYK 값 그라데이션 범위
A 00D0B6 0, 208, 182 68, 0, 40, 0 13.5%
B 00CF9C 2, 83, 165 93, 71, 9, 0 50%
C 00CF80 0, 149, 145 100, 0, 45, 15 91%

여백 및 최소 크기

여백

가시성을 보장할 수 있도록 최소한의 공간을 확보해야 합니다.

최소 크기

모바일 화면에서 30 px, 인쇄물에서 10 mm 이상이어야 합니다.

배경 색상

심볼이 모든 환경에서 높은 명시성이 유지될 수 있도록 색의 대비를 고려해 배경 색상을 선택해야 합니다. Black 배경과 CLOVA Green 배경을 적용할 수 있으며, 각 배경에 따라 사용할 수 있는 배경 색상이 다릅니다.
